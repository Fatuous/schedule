Входные данные:
	cab.txt - [№ аудитории] [вместимость] [тип]
		[№ аудитории]: только цифрами
		[тип]: пр - практическое занятие, лек - лекция, лаб - лабораторная работа
			Пример:	251 25 лаб
				327 40 пр
				446 120 лек
	
	group.txt - [название группы] [кол-во человек] [номер потока]
		Пример: group_1 30 1
			group_2 25 2
	
	lessons.txt - 	[№ потока]
			[название предмета] [тип] [кол-во часов]
		[тип]: пр - практическое занятие, лек - лекция, лаб - лабораторная работа
		Расписание для каждого потока должно быть отделено пустой строкой
			Пример: 1
				lesson_1 пр 16
				lesson_2 лек 16
				lesson_2 пр 32

				2
				lesson_1 лек 16
				lesson_2 лаб 8

Выходные данные:
	output.txt
		[название группы]
		[№ недели]
		[день недели]
			[расписание на день]

Описание классов и функций:

Класс group - хранит данные о группах
	структура timetable - хранит данные о кабинете, предмете и типе занятия
		cabinet - кабинет
		lesson - предмет
		type - тип
	name - название группы
	quant - количество человек
	flow - номер потока
	predmet - расписание занятий на каждый день


Класс cab - хранит данные о кабинетах
	number - номер кабинета
	capacity - вместимость
	type - тип
	left - количество мест в данный день на данной паре
	flow - номер потока в аудитории
	lesson - предмет в данный день на данной паре
	set_capacity() - устанавливает оставшееся количество мест


Класс lesson - хранит данные о предметах
	name - название предмета
	hours - количество часов
	type - тип
	flow - поток



cab_quant() - возвращает количество 

group_quant() - возвращает количество групп

lesson_quant() - возвращает количество предметов

push_cab() - загружает данные о кабинетах из файла в массив

push_group() - загружает данные о группах из файла в массив

push_lesson() - загружает данные о предметах из файла в массив

sort_groups() - сортировка массива групп по убыванию количества человек в них

sort_cabs() - сортировка массива кабинетов по убыванию вместимости

sort_lessons() - сортировка массива предметов по убыванию часов

create() - формирует расписание

print() - сохраняет расписание в файл
